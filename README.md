# 🏛️ AI Personal Historian (Streamlit App)

An interactive AI-powered chatbot that answers history-related questions in a structured and concise format using Google Gemini.

---

## 🚀 Features

- 💬 Chat-based interface using Streamlit
- 🧠 Powered by Gemini AI (Google Generative AI)
- 📚 Provides structured historical answers:
  - Event
  - Time Period
  - Key Figures
  - What Happened
  - Historical Impact
- 📝 Maintains chat history within the session

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Google Generative AI (Gemini)
